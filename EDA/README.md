# Exploratory Data Analysis

This folder contains the notebooks and source data used to clean, inspect, and understand the utility-consumption time series before feature engineering and model training.

## Contents

- `preprocess.ipynb` cleans the raw data, checks data quality and anomalies, enriches it with weather information, and prepares the total consumption field.
- `EDA.ipynb` explores consumption and weather distributions, trends, seasonality, correlations, and time-series behavior.
- `EDA_with_holidays.ipynb` extends the analysis with public and regional holiday information to examine its relationship with demand.
- `Utility_consumption.csv` is the source utility-consumption dataset used by the notebooks.
