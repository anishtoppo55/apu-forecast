# Feature Engineering

This folder contains the notebook and datasets used to convert cleaned utility-consumption data into model-ready features.

## Contents

- `Feature_engineeering.ipynb` creates calendar, cyclical time, weather, holiday, lag, and rolling-window features for each prediction target.
- `Utility_consumption.csv` is the input utility-consumption dataset available in this folder.
- `Utility_features_engineered.csv` is the generated feature dataset used for model training.
